🎮 Brick Breaker Deluxe 🎮

📘 Instructions

Bienvenue dans Brick Breaker Deluxe !

Objectif : Casser toutes les briques à l’aide de la balle.
Commandes :
- Mode Clavier : flèches ← → pour bouger, ↑ pour lancer, Espace pour pause.
- Mode Souris : bouge la souris pour contrôler la palette, clique pour lancer.

Skins : Personnalise le look du jeu à ton goût !
Niveaux : La difficulté augmente à chaque niveau.
Pause : Appuie sur Espace pour mettre en pause / reprendre.

Bonne chance et amuse-toi bien ! 😄
